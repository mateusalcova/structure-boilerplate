Structure Boilerplate
=====================

This is a standard framework for developing web apps.

---


###### Pattern

    .
    /-> css
    |   /-> fonts
    |   /-> sass
    |   |   |-> config.rb
    |   |   |-> fonts.sass
    |   |   |-> footer.sass
    |   |   |-> header.sass
    |   |   |-> init.sass
    |   |   |-> main.sass
    |   |  
    |   |-> fonts.css
    |   |-> footer.css
    |   |-> header.css
    |   |-> init.css
    |   |-> main.css
    |   
    /-> img
    |   |-> _blank.gif
    |   |-> favicon.ico
    |
    /-> inc
    |   |-> controller.php
    |   |-> footer.php
    |   |-> header.php
    |   |-> send-contact.php
    |   |-> smtp.php
    |
    /-> js
    |   |-> app.js
    |   |-> html5.js
    |   |-> zepto.js
    |
    /-> pages
    |   |-> home.php
    |
    |-> .htaccess
    |-> index.php
    |-> README.md (this)

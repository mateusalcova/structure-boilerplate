Structure Boilerplate
=====================

This is a standard framework for developing web apps.

---

###### Pattern

    .
    /-> css
    |   /-> sass
    |   |   |-> main.sass
    |   |  
    |   |-> main.css
    |   
    /-> img
    |   |-> _blank.gif
    |   |-> favicon.ico
    |
    /-> inc
    |   |-> controller.php
    |   |-> footer.php
    |   |-> header.php
    |
    /-> js
    |   |-> app.js
    |   |-> jquery.js
    |
    /-> pages
    |   |-> home.php
    |
    |-> .htaccess
    |-> index.php
    |-> README.md (this)
